Start and Stop

You can stop and start the simulation process by clicking the buttons at top left part of the control panel. The same operations are possible by the menu items Start and Stop in the Projection menu in the menu bar, whose shortcut keys are ⌘G and ⇧⌘G respectively. When you stop the simulation, the projection swarm disappears suddenly, but it appears gradually by Fade in time when it starts again. You can change this duration by alternate the value displayed at middle right part of the control panel, as shown in a figure in the later section of Appearance.

Full Screen Mode

The switch at the top middle is to turn the full screen mode on and off. If your computer has more than one screens, the control panel can stay visible when the projection and the control panel are placed in the different screens. The switch is effective to turn the full screen mode off. The menu item Full Screen in the Projection menu in the menu bar has the same function to switch the mode. The shortcut key of this menu item is ⌘F, that is effective to turn the mode off even when the control panel is hidden behind the full screen projection. ESC key also has the same effect to exit from the full screen mode.

Selecting screen for full screen mode

In usual setups, a projector is attached to the video output port of the computer as the secondary screen. A typical example is a HDMI connection with MacBook Pro without screen mirroring. In this case, you don't need to make any changes for screen settings since this software selects the secondary screen for full screen mode. In case your computer has more than two screens, it is helpful to specify the screen to be used for full screen mode, by a popup button at the bottom part of control panel. The menu attached to this button contains the items entitled screen names.

Selecting Projection Mode

Using the popup button at the top right part of control panel, you can choose one of the projection modes; Normal, Adjustment, Masking, Attractant, and Repellent. The menu items of the same titles in the Projection menu in the menu bar are also available to change the mode. Each mode displays as follows.

  • Normal: swarm animation for the real performance.
  • Adjustment: detected area and grid for geometric adjustment.
  • Masking: to edit the masking bitmap to avoid interference by noise.
  • Attractant, Repellent: monitoring the distribution of those virtual chemicals to control the swarm behaviour.
