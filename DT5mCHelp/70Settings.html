Settings Top Prev

The window shown above appears when you click the menu item Settings ... in DT5mC menu in the menu bar. You can change parameters for communication with the tracker and simulation frame rate.

  1. Reveiver's Port Number: port number to receive packets from tracker. The default value is 9003.
  2. Bitmap Size: Automatic: if checked, the system calculates the image size from the packet size assuming the aspect ratio is 16:9 or 4:3. An error message is shown if it failed to determine the size. If this check box is not checked, it uses the numbers in Width and Height text fields. It shows an error message when the packet size does not fit with the image size.
  3. Simulation steps interval:: magnitude of interval between simulation steps. This parameter should be adjusted so that the simulation frame rate becomes 60 FPS or as similar value as possible. The real frame rate is displayed in the bottom part of the control panel.
