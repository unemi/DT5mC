Each agent, a member of swarm, is drawn as line segments connecting a number of points it passed. You can modify the appearance of swarm by the following five parameters accompanying the number of agents and agent colour:

  1. Length: total length of each agent. The tail segment might be shorten if it would be longer to connect to the passed point.
  2. Weight: thickness of line segment.
  3. Max Opacity: opacity value when the agent is on the detected target.
  4. Min Opacity: opacity value at the position where the attractant density is zero. The default value is zero. When you give it higher value, the agents at positions of low attractant density become visible, that enables to observe how the agents explore in the space by high speed movement.
  5. Opacity Gradient: steepness of opacity value corresponding to the attractant density.
